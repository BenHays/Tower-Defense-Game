# Core Loop

## Live now

- Day is untimed and starts with two day actions.
- Clearing a tree, building a defense, and upgrading a launcher use the full day.
- Repairing and placing Scout use one action.
- The player ends the day; night combat is automatic. Scout returns before the next day begins.

## Proposed next

- Keep the player decision focused on expansion, placement, and research. Do not add direct night combat controls.
- Replace the starting teepee with a Level 1 shelter site. **Construct shelter** is the only enabled first-day action; it costs 0 wood, spends the full day, and unlocks every other control after completion. **End day** stays disabled until the teepee exists.

## Open questions

- Whether the first shelter build should automatically advance to night or leave the newly enabled **End day** button as the player's next deliberate choice.

## Decisions made

- There is one authored map. Seeds vary encounters, not terrain.
- A level message should only explain the current objective or a new unlock.
- The teepee is the sole zero-cost building, but it is earned through the forced opening action rather than granted on map load.
