# Core Loop

## Live now

- Day is untimed and starts with two day actions.
- Clearing a tree, building a defense, and upgrading a launcher use the full day.
- Repairing and placing Scout use one action.
- The player ends the day; night combat is automatic. Scout returns before the next day begins.

## Proposed next

- Keep the player decision focused on expansion, placement, and research. Do not add direct night combat controls.

## Open questions

- None for the current MVP slice.

## Decisions made

- There is one authored map. Seeds vary encounters, not terrain.
- A level message should only explain the current objective or a new unlock.
